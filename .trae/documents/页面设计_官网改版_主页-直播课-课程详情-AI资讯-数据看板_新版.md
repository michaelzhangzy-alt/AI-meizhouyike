# 页面设计说明（桌面端优先）

## 全局规范
### Layout
- 桌面端（>=1200px）：最大内容宽度 1120px，左右留白自适应；12 栅格；区块间距 48px。
- 平板（768-1199px）：最大宽度 720-960px；区块间距 32px。
- 移动端（<=767px）：单列堆叠；区块间距 24px；按钮与表单全宽。

### Meta（通用）
- title：`{页面名} | {品牌名}`
- description：各页面 120-160 字，包含“直播课/课程/AI资讯”等关键词。
- Open Graph：`og:title`、`og:description`、`og:type=website`、`og:image`（统一品牌图）。

### Global Styles（Design Tokens）
- 背景：`--bg: #0B1220`（深色）或 `#FFFFFF`（浅色，二选一保持一致）
- 主色：`--primary: #2563EB`；强调：`--accent: #22C55E`；警示：`--danger: #EF4444`
- 文本：`--text: #0F172A`（浅底）/ `#E2E8F0`（深底）
- 字体层级：H1 40/48，H2 28/36，H3 20/28，正文 16/24，注释 13/20
- 按钮：Primary（实心主色）、Secondary（描边）、Link（文字）；hover 亮度 +6%，disabled 40% 不透明
- 卡片：圆角 12px；阴影轻量；列表 hover 高亮边框

### 通用组件
- TopNav：Logo、菜单（本周直播课/AI资讯）、右侧主 CTA（报名/咨询）
- Footer：公司信息、联系信息、声明（可选）
- Card：直播课卡片/课程卡片/资讯卡片
- Form：最小字段收集（手机号或微信必填），提交后 toast + 成功页文案

## 页面1：主页
### Layout
- 结构：纵向分区（Hero → 入口区 → 精选内容 → 转化区 → Footer），使用 CSS Grid（桌面两列，移动单列）。

### Meta
- title：`官网首页 | {品牌名}`
- description：突出“本周直播课 + 课程体系 + AI资讯”。

### Page Structure
1) 顶部导航（固定或吸顶）
2) Hero 首屏（左文案右插图/人物/课程图）
3) 快捷入口区（两张大卡：本周直播课、AI资讯）
4) 精选内容区（最新直播课 1-3 条 + 最新资讯 3 条）
5) 转化区（强 CTA + 简表单入口）

### Sections & Components
- Hero：标题（1 行价值主张）+ 2 行说明 + 主按钮“查看本周直播课” + 次按钮“查看AI资讯”。
- 快捷入口卡：显示模块说明、数量徽标（如“本周 3 场”）。
- 精选列表：卡片网格；卡片包含标题/时间/标签；点击进入对应详情。
- 转化区：强调“报名/咨询”；按钮触发表单弹窗或跳转表单区。

## 页面2：本周直播课
### Layout
- 桌面：左侧筛选/周切换（可简化为顶部周选择），右侧列表；Flex + 列表虚拟滚动可选。

### Meta
- title：`本周直播课 | {品牌名}`
- description：本周直播课时间表与报名入口。

### Page Structure
1) 顶部：周范围（如 2026-W05）+ 状态说明
2) 直播课列表（按时间排序）
3) 固定 CTA 区（移动端底部固定“立即报名/咨询”）

### Sections & Components
- LiveSessionCard：时间、标题、状态、按钮“查看详情/报名”。
- EmptyState：本周无直播课时展示引导（返回主页/查看AI资讯）。

## 页面3：课程详情
### Layout
- 桌面：两列（左正文，右侧信息栏/报名卡）；移动端改为信息栏置顶卡。

### Meta
- title：`课程详情：{课程名} | {品牌名}`
- description：包含课程亮点、适合人群、报名引导。

### Page Structure
1) 面包屑（主页 > 本周直播课 > 课程详情）
2) 标题区（课程名、摘要、标签）
3) 内容区：亮点/适合人群/你将获得
4) 侧栏：时间信息（若为直播课承接）、主 CTA（报名/咨询）

### Sections & Components
- InfoSidebar：展示关键字段 + 主按钮；按钮点击后打开表单弹窗。
- RelatedSessions（可选）：同课程的其他场次入口（若存在）。

## 页面4：AI资讯（列表+详情）
### Layout
- 列表：桌面 3 列卡片网格；详情：单列阅读（最大宽 760px）。

### Meta
- 列表 title：`AI资讯 | {品牌名}`
- 详情 title：`AI资讯：{标题} | {品牌名}`

### Page Structure
- 列表页：顶部筛选（类型：原创/外链导读；标签）+ 卡片列表 + 分页/加载更多
- 详情页：标题/元信息（发布时间/标签/类型）+ 正文（原创）或导读（外链）+ 外链按钮 + 关联推荐

### Sections & Components
- FilterBar：类型切换（Tab）+ 标签下拉。
- ExternalGuideBlock：导读要点（3-5 条）+ 外链按钮（新开标签页）。
- RelatedList：3-5 条推荐卡。

## 页面6：数据看板（GA4 漏斗 + 日常 5 报表）
### Layout
- 桌面：顶部筛选条 + 下方两列（左：漏斗/趋势，右：5 报表与结论）；Grid 8/4 或 7/5。

### Meta
- title：`数据看板 | {品牌名}`
- description：GA4 漏斗与日常 5 报表复盘。

### Page Structure
1) 顶部筛选：日期范围、渠道（可选）、刷新/导出
2) 漏斗卡：步骤人数、转化率、掉点提示
3) 5 报表区：按报表 1-5 卡片化展示（摘要 + 明细表）
4) 闭环记录：当日结论（文本域）+ 行动项（表格：事项/负责人/截止）+ 保存

### Interaction States
- Loading：骨架屏（漏斗与表格）。
- Empty：无快照时提示“生成/拉取今日报表”。
- Export：导出按钮支持 CSV；复制摘要支持一键复制。

### Analytics（埋点表现层约束）
- 关键按钮（查看本周直播课/查看详情/报名/提交/外链跳转/导出）统一具备可追踪属性（data-analytics-id）。
