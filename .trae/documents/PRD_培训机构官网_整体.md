# PRD｜培训机构官网（整体：阶段1 + 阶段2 + AI工具）

## 1. 背景与目标

本项目为「公开课引流 + 内容沉淀 + AI 效率工具 + 后台运营提效」的一体化官网。

### 1.1 业务目标
- 通过公开课/师资展示降低用户决策成本
- 用极简留资表单获取高质量线索
- 用课程与资讯内容持续获取自然流量（SEO）与复访
- **通过 AI 工具（如运势分析、文案生成）增加用户粘性与趣味性**
- 通过后台工具降低运营发布与销售跟进成本

### 1.2 成功指标（可量化）
- 留资转化：报名弹窗打开率、提交成功率、每周新增线索数
- 内容增长：每周发布课程/资讯数量、资讯阅读量（views）、文章复访率
- **AI 互动：AI 工具的使用频次、生成结果的复制率**
- 运营效率：线索导出耗时、内容发布耗时、页面更新是否“无需改代码”

## 2. 用户与场景

### 2.1 用户角色
- **访客（在校学生/职场新人）**：浏览公开课/师资/资讯；使用 AI 工具娱乐或辅助工作；完成报名留资；转发分享
- **运营/教务**：维护课程、资讯、讲师、站点文案；配置微信分享
- **销售/顾问**：查看线索、导出 Excel、跟进转化

### 2.2 关键场景
1. 学生通过朋友圈/群聊/搜索进入首页 → 了解本周公开课 → 弹窗报名
2. 学生通过资讯列表阅读干货 → 查看详情 → 产生信任 → 预约/报名
3. **用户出于好奇使用“科学算运势” → 生成趣味报告 → 截图分享到社交媒体**
4. **运营人员使用“小红书文案生成器” → 快速产出推广文案 → 发布到外部平台**
5. 运营在后台发布新课程/新文章 → 配置分享卡片 → 引导传播
6. 销售在后台筛选并导出线索 → 跟进转化

## 3. 范围与阶段划分

### 3.1 阶段1（MVP）范围
- 首页展示（公开课/师资）
- 报名弹窗留资（姓名、手机号、微信号可选）
- 关于我们（基础介绍 + 联系方式）

### 3.2 阶段2（运营提效）范围
- 后台管理系统（CMS）：课程、资讯、师资、站点内容管理
- 线索管理（Leads）：查看、搜索、导出
- 微信分享配置
- 会员与社群管理

### 3.3 阶段3（AI 赋能）范围（v1.0 新增）
- **AI 工具箱**：集成 Coze 智能体能力
- **科学算运势**：基于用户输入的八字命理分析
- **小红书文案生成器**：辅助运营快速产出爆款笔记

## 4. 功能需求（整体）

### 4.1 前台功能
#### 4.1.1 首页（`/`）
- 本周公开课：展示“主推课程 + 近期预告”
- 师资力量：展示讲师列表
- 报名入口：多处触发弹窗，统一留资

#### 4.1.2 往期课程（`/courses`、`/courses/:id`）
- **列表页**：展示所有“已发布”的课程（卡片式布局）
- **详情页**：课程头图、核心信息、富文本介绍、侧栏讲师、底部报名
- **本周直播（`/weekly-class`）**：自动跳转到最近一期未开始的课程详情页

#### 4.1.3 AI 资讯（`/news`、`/news/:id`）
- 列表：展示已发布文章（支持**外链导读**模式，点击直接跳转外部 URL）
- 详情：富文本正文渲染、图片展示、侧栏推荐
- 阅读量：进入详情页时更新 `views`

#### 4.1.4 AI 效率工具（`/ai-tools`）
- **工具列表页**：展示所有可用的 AI 工具卡片
- **科学算运势**：
    - 输入：姓名、性别、出生时间、地点
    - 输出：流式生成的命理分析报告（包含五行、流年运势等）
    - 体验：支持“打字机”效果，无延迟，无乱码
- **小红书文案生成器**：
    - 输入：主题、关键词
    - 输出：符合小红书风格（Emoji、分段、语气）的爆款文案
    - 功能：一键复制结果

#### 4.1.5 关于我们（`/about`）
- 发展历程时间轴
- 会员权益展示（后台配置）

### 4.2 后台功能（`/admin/*`）
- **仪表盘**：核心指标概览
- **报名管理**：线索列表、Excel 导出
- **课程/资讯/师资管理**：全功能 CRUD、富文本编辑、图片上传
- **微信分享配置**：自定义分享卡片（标题/描述/图片）
- **会员管理**：付费名单导入、群二维码配置

## 5. 非功能需求

### 5.1 体验
- **响应式设计**：完美适配 PC 与 移动端
- **流式响应（Streaming）**：AI 工具必须支持 SSE 流式输出，首字延迟（TTFT）< 1秒
- **视觉风格**：简约极客风（Minimalist Cyber），强调专业与科技感

### 5.2 安全与合规
- 线索数据权限严格控制（仅后台可读）
- AI 接口通过后端代理转发，隐藏 API Key
- JSON 过滤机制防止 AI 输出系统日志
