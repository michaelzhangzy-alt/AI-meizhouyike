# 页面设计（Desktop-first）

## 全局规范
### Layout
- 桌面优先：内容最大宽度 1120–1200px，居中；两侧留白。
- 响应式：
  - ≥1024px：双列/多列卡片布局（公开课、师资）。
  - 768–1023px：两列卡片。
  - <768px：单列堆叠，按钮全宽，弹窗改为底部抽屉式或全屏弹层。
- 间距：8px 基准（8/16/24/32）。

### Meta Information（默认）
- Title：培训机构｜本周公开课报名
- Description：查看本周公开课与师资，立即报名获取课程信息。
- Open Graph：og:title/og:description/og:type=website/og:url/og:image（占位图）。

### Global Styles（Design Tokens 建议）
- 背景：#0B1220（深色）或 #FFFFFF（浅色，二选一保持一致）
- 主色（CTA）：#2563EB
- 强调色：#F59E0B
- 文字：主文 #111827，次文 #6B7280
- 字体层级：
  - H1 40/48
  - H2 28/36
  - H3 20/28
  - Body 16/24
  - Small 14/20
- 按钮：
  - Primary：主色底白字；hover 加深 8%；disabled 降低透明度。
  - Secondary：描边/浅底；hover 轻微底色。
- 链接：主色 + underline on hover。

## 页面 1：首页（/）
### Layout
- 顶部导航固定在页面顶部（可选 sticky）。
- 主体为纵向堆叠区块：Hero → 本周公开课 → 师资 → 页脚。

### Meta Information
- Title：本周公开课｜立即报名
- Description：本周公开课时间与亮点、核心师资介绍，一键报名。

### Page Structure
- 结构：单列主轴 + 内部卡片网格（公开课/师资）。
- 关键交互：所有“立即报名”打开同一报名弹窗（带来源 source）。

### Sections & Components
1. 顶部导航（NavBar）
   - 左：Logo（点击回首页）
   - 右：菜单项（首页、关于我们）；移动端汉堡菜单。

2. Hero 主视觉区（Hero）
   - 标题：一句话价值主张（例如“本周公开课，带你快速上手XX”）
   - 副标题：补充说明（适合人群/学习收益）
   - 主按钮：立即报名（Primary）
   - 次按钮（可选）：了解师资/查看公开课（锚点跳转）

3. 本周公开课（OpenClassThisWeek）
   - 区块标题 + 简短说明
   - 主推卡（Featured Card）：
     - 课程名、时间、地点/线上、3 个亮点要点
     - CTA：立即报名
   - 列表卡（Card Grid）：最多 3–6 个（MVP 可先 1–3 个）
     - 关键信息：标题、时间、适合人群
     - CTA：立即报名

4. 师资展示（Teachers）
   - 卡片网格：头像（占位）、姓名、头衔/领域、1–2 句亮点
   - 每张卡 CTA：立即报名

5. 报名弹窗（LeadModal）
   - 触发：点击任意“立即报名”
   - 容器：桌面居中 Modal（宽 420–480px）；移动端底部抽屉/全屏
   - 字段（极简）：
     - 姓名（必填）
     - 手机号（必填，格式校验）
     - 微信（选填）
     - 意向课程（选填，可从触发位置预填/带入公开课名称）
   - 按钮：提交（Primary）、取消/关闭（Secondary）
   - 状态：
     - 提交中：按钮 loading
     - 成功：提示“已提交，我们将尽快联系你”+ 关闭按钮
     - 失败：错误提示 + 重试
   - 细节：
     - 支持 Esc 关闭（桌面）
     - 点击遮罩关闭（可选，避免误触可关闭此能力）

6. 页脚（Footer）
   - 基础信息：机构名、版权、联系方式（最少一项）
   - 链接：关于我们
